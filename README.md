## Описание Задания 1

Составить план тестирования карточки товара для выявления XSS и SQL-инъекций.

В карточке товара находятся следующие поля ввода:

- цена — числовое поле;
- остаток товара на складе — числовое поле;
- название товара;
- описание товара.
